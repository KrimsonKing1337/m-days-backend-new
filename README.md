# m-days-backend

Этот репозиторий - backend для проекта Монохромные дни.

Монохромые дни - это апсайклинг проект. Для того, чтобы дать старым устройствам возможность снова быть полезными, а не пылиться где-то на полке (или ещё хуже на свалке). Устройство превращается в красивую умную фоторамку с отборными фотографиями со всего света. 

На данный момент приложение работает в IE3+ браузерах, но это не предел. Цель - заставить работать на ещё более старых браузерах и устройствах.

Frontend можно найти здесь:\
